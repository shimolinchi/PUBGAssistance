# 🗺️ PUBG 地图助手 (PUBG Map Assistant)

这是一个轻量级、安全且现代化的 PUBG (绝地求生) 桌面悬浮地图辅助工具。程序基于纯 Python 构建，通过原生的透明悬浮窗技术在屏幕上方叠加点位信息（如：固定刷车点、密室、滑翔机等），**不读取任何游戏内存、不修改游戏文件**，最大程度保证账号安全。

## ✨ 核心特性

* 🎮 **纯净悬浮层**：利用 Tkinter 构建真正的透明穿透窗口，点位悬浮于游戏画面之上，鼠标点击可直接穿透，不影响任何游戏内操作。
* 🖱️ **极客全局快捷键**：告别传统的键盘快捷键冲突。**同时按下鼠标 `左键 + 中键 + 右键`** 即可无缝呼出/隐藏地图点位。
* 🎯 **智能无缝标点**：在悬浮窗开启状态下，**单按鼠标右键**在游戏内大地图进行标记的同时，程序会自动隐藏悬浮窗，体验极其流畅。
* 🎨 **现代化 UI 界面**：摒弃传统 Tkinter 的古老外观，采用 B样条曲线 (B-spline) 绘制底层平滑连续的圆角交互按键。
* 🧰 **内置点位抓取工具**：附带逆向坐标计算程序，可直接在游戏大地图中点击获取绝对归一化坐标，方便后续无限扩展。

## 🛠️ 环境依赖

本项目仅依赖 Python 内置库以及轻量级的全局外设监听库 `pynput`。

**要求：** Python 3.7 或更高版本。

**安装依赖：**
```bash
pip install pynput
```
**打包**

```
nuitka --onefile --plugin-enable=tk-inter --windows-disable-console --windows-icon-from-ico=icon.ico main.py
```
