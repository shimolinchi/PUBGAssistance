# PUBG Tactical Hub (绝地求生战术助手)

基于纯视觉识别 (OpenCV) 与底层键鼠模拟 (pynput) 开发的 PUBG 战术助手。
本项目不进行任何内存注入或修改游戏文件，纯基于屏幕截图分析与数学几何计算，提供从测距、投掷到特种武器弹道计算的全方位战术支持。

## 核心特性 (Features)

* **纯视觉安全架构：** 基于 mss 高频截图与 OpenCV 形态学处理，零内存读取。
* **配置驱动 (Data-Driven)：** 所有弹道数据、快捷键、UI坐标均剥离至 config.json，无需修改源码即可适配新版本或新分辨率。
* **底层快速触发机制：** 可为游戏内操作自定义绑定快捷键，如手雷自动瞬爆、开启测距显示等
* **高度模块化：** 主控台与各武器助手完全解耦，随时启停，互不干扰。

## 功能模块说明 (Modules)

* **小地图实时测距 (minimap_radar.py)：** 动态监控小地图标点，实时计算各颜色标点到玩家的真实物理距离。
* **大地图快照测距 (largemap_radar.py)：** 采用“单次快照 (One-Shot)”架构，按下快捷键瞬间截取大地图并计算距离，数据缓存固定，极低 CPU/内存占用。
* **地图资源点位 (map_assistant.py)：** 归一化坐标数据库，支持在各种分辨率下精准显示刷车点、密室、飞机航线等战术资源。
* **迫击炮测距模块 （mortar_assistant.py）:** 通过小地图标点获得水平距离，垂直测高模块获得高低位置，计算出迫击炮打击实际距离
* **武器自动识别 （weapon_detector.py）:** 通过屏幕下方的武器图标进行模板匹配，自动识别使用武器（仅限使用到的武器模块）
* **VSS自动弹道计算 （vss_assistant.py）:** 通过小地图标点测距，获得准星上移距离
* **十字弩自动弹道计算 （crossbow_assistant.py）:** 通过小地图标点测距，获得准星上移距离
* **火箭筒自动弹道计算 （rocket_assistant.py）:** 通过小地图标点测距，获得准星上移距离
* **手雷自动瞬爆与抬高计算 （throwables_assistant.py）:** 通过小地图标点测距，获得准星上移距离，以及瞬爆时间，通过绑定按键（默认V）实现自动瞬爆

## 使用方法

+ 使用前需要进行小地图与大地图的校准（如果你的显示器不是1080p的情况下），使用程序窗口上方的两个按键即可。分别点击**放大后**的小地图与大地图的两个对角顶点即可完成校准。
+ 游戏改成繁体中文，接着色盲模式需选成绿色盲，就是打人血是浅蓝色那个
+ 进入程序后，按下 N 或者 Shift + Ctrl + Space 可开启显示层。
    + 设计这两个默认按键的原因是：N是游戏中默认方法小地图的按键，本程序的测距完全依赖**放大后**的小地图，因此必须开启小地图放大才能进行测距。但一旦进入游戏之前误触了N，可以使用 Shift + Ctrl + Space 进行调序。
+ 若开启了显示层，拾取对应的武器会自动识别（包含手雷、火箭筒、十字弩、VSS），迫击炮默认正常运行（因为不与其他武器显示层重叠）
+ 开启显示层后，必须要有敌人位置精确标点才能计算距离，且需要在小地图范围之内
+ 本助手中火箭筒的有效射程为 0~150m，手雷为 0~50m，VSS为 100~350m（小于100m无需抬高，故而不显示），十字弩为 0~350m
+ 迫击炮测距有两层显示，在右侧小地图上方，第一层为大地图测距获得（需要大地图中手动触发），第二层为小地图实时测距。四个颜色分别代表四个标点距离，颜色深浅代表置信度。若无标点则为最深，只有水平距离为偏按，水平与垂直标点都有为最亮，此时实际发射距离计算最准。
+ 大地图测距需要打开大地图，放大到第三等大小，按下快捷键 （默认Ctrl + Shift + Tab），会自动测出标点到玩家的四个位置，并同步到右侧第一行显示层上。

## 配置文件修改方法

配置文件中包含了如下项：

+ minimap_rect：小地图标定数据，如果嫌标定不准直接改也行
+ minimap_colors：小地图标点检测四个颜色通道。如果修改了其他色盲模式，就需要修改这些
+ map_rect：大地图标定数据，side是显示器窄边，因为大地图是顶着窄边放的
+ map_1km_pixels：大地图缩放到第三级后1km对应的像素数，大地图测距的时候用的。游戏中1km对应一大格
+ elevation_colors：垂直测高模块检测的四个颜色通道，如果修改了其他色盲模式，就需要修改这些
+ elevation_rect：垂直测高模块区域，就中间一小条，迫击炮方向对准目标时候用，大概大概就行，小一点可以节省计算资源
+ hotkeys：设置的快捷键，直接改也行
+ **config：所有武器的距离到抬高高度等的映射
+ map_data：大地图标点点位，包括飞机、密室、固刷车等


> **提示：** 使用 ratio (比例 0.0 ~ 1.0) 代替绝对像素坐标，使得你在 1080P、2K 或 4K 显示器下切换时，无需重新标定数据！

## 安装与运行

### 环境依赖
请确保已安装 Python 3.10.20，并安装以下依赖库：
```bash
    pip install opencv-python numpy mss pynput
```
### 运行程序

windows打包

```bash
nuitka --onefile --plugin-enable=tk-inter --windows-disable-console --windows-icon-from-ico=icon.ico --include-data-files=icon.ico=icon.ico main.py
```

### 快捷键配置
程序内置了 UI 修改功能，你可以在主界面直接点击修改快捷键：
* **显示开关：** 默认为 <ctrl>+<shift>+<space> 或硬件保留键 n。
* **大地图测距：** 默认为 <ctrl>+<shift>+m。
* **手雷瞬爆：** 默认为 v (按下准备，松开投掷)。
