# PUBG 综合战术助手 (PUBG Tactical Assistant)

![Python Version](https://img.shields.io/badge/Python-3.8%2B-blue.svg)
![License](https://img.shields.io/badge/License-MIT-green.svg)

一款轻量级、安全且纯基于数学计算与外设监听的 PUBG（绝地求生）战术悬浮窗助手。本项目集成了**地图固定点位（刷车点、密室、熊洞等）显示**与**迫击炮精准测距**两大核心功能。

**安全声明：** 本程序为**纯物理级悬浮窗外挂（Overlay）**，**不读取任何游戏内存，不截取游戏画面（无屏幕识别代码）**，仅通过监听全局鼠标与键盘事件并进行几何运算来实现功能，极大地降低了封号风险。

---

## 核心功能 (Features)

* **全地图战术点位全息投影**
    * 支持艾伦格、米拉玛、泰戈、荣都、维寒迪、帝斯顿等全尺寸大地图。
    * 直观显示：固定刷车点、飞机刷新点、密室、撬棍房、熊洞、安全门等高价值战略目标。
    * 支持通过快捷键快速切换地图，并在屏幕左上角提供战术 HUD 显示当前地图名称。
* **极速迫击炮测距系统**
    * 基于屏幕像素系与游戏网格系统的数学换算。
    * 提供【1km】与【100m/200m】独立标定按键，一键将屏幕像素转换为实际距离。
* **三档动态预设标尺 (Zoom Levels)**
    * 支持独立保存 3 个游戏缩放档位的标尺比例，实战中按 `+` / `-` 键瞬间切换，无需重新标定。
* **防遮挡战术 HUD (Heads Up Display)**
    * 屏幕左上角：显示当前加载的地图名称。
    * 屏幕左下角：实时显示当前预设标尺档位及 1km 对应的像素数值，数据状态一目了然。
* **纯异步无冲突单状态机**
    * 彻底解决功能冲突，地图模式与测距模式无缝互斥切换，操作如丝般顺滑。

## 环境依赖 (Requirements)

本项目基于 Python 开发，依赖极少，运行轻快。

1.  **Python 3.8 或更高版本**
2.  **tkinter** (通常随 Python 标准库自带)
3.  **pynput** (用于全局接管键盘与鼠标事件)

### 安装依赖包：
```bash
pip install pynput
```
### 打包

```bash
nuitka --onefile --plugin-enable=tk-inter --windows-disable-console --windows-icon-from-ico=icon.ico main.py
```
## 快速启动 (Quick Start)

1. 将项目文件（`main.py`）克隆或下载到本地。
2. 在项目目录下打开终端（或命令提示符）。
3. 运行主程序：
   ```bash
   python main.py
   ```
4. 程序启动后，进入游戏，使用游戏设置将游戏窗口调整为 **无边框全屏 (Borderless)** 或 **窗口化 (Windowed)**。*(注意：由于操作系统限制，独占全屏模式下无法显示透明悬浮层)*。

## 操控手册 (Controls & Hotkeys)

本程序专为高强度电竞环境设计，所有操作均可通过外设快捷键完成，让你在交火中保持专注。

### 地图显示控制
| 操作 / 快捷键 | 功能描述 |
| :--- | :--- |
| **`左键` + `中键` (同时按)** | 呼出 / 关闭 地图战术点位显示。 |
| **`右键` (单按)** | 在显示地图时，清除屏幕上的点位（隐藏悬浮窗）。 |
| **按住 `Alt` + `右键`** | 游戏中用于标点/画路线，此时程序**不会**清除地图点位。 |
| **按住 `Alt` + `+` 或 `-`** | 快速切换当前显示的大地图（如从艾伦格切到米拉玛）。 |

### 迫击炮测距与标定
| 操作 / 快捷键 | 功能描述 |
| :--- | :--- |
| **`Tab`+`Caps`+`Shift`+`Ctrl`** | **一键启动迫击炮测距模式**。此时屏幕清空，等待你点击起点和终点。 |
| **鼠标 `左键`** | 测距/标定状态下：选取地图上的点。连续点两下即可出距离结果。 |
| **鼠标 `右键`** | 在任何测距或标定阶段，单按右键直接**取消并返回待命状态**。 |
| **键盘 `+` 或 `-`** | 切换预设的缩放标尺档位 (1档 / 2档 / 3档)。 |

### 全局紧急控制
| 操作 / 快捷键 | 功能描述 |
| :--- | :--- |
| **鼠标单独按 `中键`** | 紧急状态复位，无论程序在干什么，强制退回“待命”状态。 |

## 配置与数据持久化 (Configuration)

程序会在根目录下自动生成 `mortar_config.json` 配置文件。
该文件用于持久化保存你的**当前缩放档位**以及**各个档位对应的标尺比例（像素数）**。
* 即使关闭程序重新打开，你之前的标定数据依然存在。
* *建议：开局在跳伞前，花 5 秒钟标定好 3 个你最常用的 M 键地图缩放档位，整局游戏即可畅享秒测距。*

## 点位自定义 (Customization)
如果你想修改地图点位的颜色或粗细，可以在主界面的【点位大小】中选择（小/中/大）。
如果需要添加新的特殊位置，只需在代码头部的 `MAP_DATA` 字典中以 `(-1.0 到 1.0)` 的归一化坐标录入即可。
