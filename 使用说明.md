# PUBG 战术助手 使用说明
---

## 1 下载与运行

### 1.1 获取程序
- 仅支持 windows 11 操作系统，windows 10 经过测试会出现显示bug
- 完整使用助手需要有一个.exe文件（可运行程序），一个templates文件夹（存放识别用模板）、一个config.json文件（保存配置参数）。

### 1.2 运行程序
- 双击 `TacticalHub.exe`。
- 如果弹出安全警告，请选择“仍要运行”，如果被windows系统拦截，需要到设置中允许运行。
- 程序启动后会显示一个 250x400 的小窗口，同时屏幕左下角会出现一个半透明的状态栏。

### 1.3 游戏设置要求（非常重要）
- **显示模式**：将游戏设为**全屏模式**。
- **分辨率**：1920x1080（其他分辨率需要额外校准，也可能出现识别问题，需要修改模板）。
- **游戏语言**：必须是 **中文**（简中或繁中均可）。
- 游戏中开火按键设置为**end**，或者若想改成其他按键需要到配置文件中修的```fire_key```参数。
- 关掉其他鼠标驱动程序

如果不满足以上任意一条，大部分功能将无法正常工作。

## 2 第一次使用前的准备（必做）

助手需要知道游戏里各个区域在屏幕上的位置，所以第一次使用前必须**校准**一次。

**怎么校准？**
1. 进入游戏训练场或实际对局（建议在装备栏界面、大地图界面、小地图打开界面各截一张全尺寸图像，用于校准识别区域）。
2. 切回助手窗口，点击 **校准区域** 选项卡，会看到很多按钮，例如“校准小地图”、“校准武器栏”、“校准武器1名称”等。
3. 按顺序点击每个按钮，屏幕会冻结并出现一个可以拖拽和缩放的方框。
4. 将该方框恰好框住对应的游戏元素
5. `鼠标左键`拖动选择区域，或按 `鼠标右键` 取消。
6. 所有项目都校准一次后，关闭助手重新启动，校准数据会自动保存。
7. 若选取的不是绿色盲的话，区域校准完后，需要校准地图点位色域，1~4号标点的HSV颜色替换到配置文件中的```pnt_colors```参数
8. 若是其他分辨率的显示器，且识别效果不理想的话，需要修改templatex里面的模板文件，weapon内部的截图更换为对应武器握在手中的截图，需要最下方的武器图标背景为纯黑。

**最少需要校准的项目**（否则核心功能无法使用）：
- 小地图
- 大地图
- 武器栏
- 武器1名称 / 武器2名称、编号数字
- 1km长度（大地图上）

**调整标点色域**

根据你的色盲颜色，调整标点的```pnt_colors```参数，推荐参数如下：

无色盲：

```json
"pnt_colors": {
    "Yellow": { "lower": [28, 150, 160], "upper": [32, 255, 255], "hex": "#E3D43C" },
    "Orange": { "lower": [8, 160, 160], "upper": [12, 255, 255], "hex": "#B3500D" },
    "Blue": { "lower": [99, 120, 160], "upper": [103, 255, 255], "hex": "#1A3EA3" },
    "Green": { "lower": [60, 150, 120], "upper": [64, 255, 255], "hex": "#109166" }
}
```

绿色盲：
```json
"pnt_colors": {
    "Yellow": { "lower": [26, 150, 160], "upper": [30, 255, 255], "hex": "#E3D43C" },
    "Orange": { "lower": [10, 160, 160], "upper": [14, 255, 255], "hex": "#B3500D" },
    "Blue": { "lower": [110, 120, 160], "upper": [114, 255, 255], "hex": "#1A3EA3" },
    "Green": { "lower": [78, 150, 120], "upper": [82, 255, 255], "hex": "#109166" }
}
```

其余项目（垂直测高、配件等）可以暂时不校准，但对应的功能会失效。

## 3 主界面速览

助手窗口有四个选项卡：

| 选项卡 | 作用 |
|--------|------|
| 地图点位 | 选择你玩的地图，以及标点大小。 |
| 启动助手 | 所有辅助功能的开关，包括武器检测、瞄准辅助、压枪、特殊武器助手（自动识别，不用手动打开）。 |
| 校准区域 | 上述的校准功能。 |
| 按键设置 | 修改快捷键。 |

**全局快捷键**（任何时候都有效）：
- `Home`：隐藏或显示助手主窗口。
- `N`或`shift+ctrl+空格`：快速开关“瞄准辅助”（N不可修改，因为该按键是游戏中开关小地图按键，需与游戏中测距显示同步,若不同步可使用`shift+ctrl+空格`强制同步）。
- `F2`：开启武器检测，相当于程序总开关，不启动则无法使用任何功能
- `shift+ctrl+tab`：压枪开关，打开武器栏自动识别枪械与配件，关闭武器栏后根据手中枪械自动计算压枪力度。
- `v`：手雷自动拉环计时抛出（自动瞬爆）
- `Q`/`E`：C4助手更换目标标点颜色
- `F1`：大地图测距，按下后鼠标左键玩家位置，可自动计算到四个标点的距离
- `鼠标左键+中键`：打开大地图点位，右键可取消

## 4 快速开始（核心流程）

### 4.1 开启武器检测
- 本助手任何功能都基于识别到的武器，因此若要使用本助手中任何功能，都需要开启此开关。
- 程序运行时默认开启。若不小心关闭，则在“启动助手”选项卡中，点击 **开启武器检测** 按钮，或按下`F2`快捷键开启
- 开启后，按下tab（或者你自定义的按键，则需要在按键设置中修改）开启武器栏，会自动识别目前的两个主武器以及装备的配件。

### 4.2 开启瞄准辅助（显示层）
- 点击 **“开启瞄准辅助”** 按钮，或者按 `N` 键，或者`shift+ctrl+空格`。
- 开启后，屏幕上会出现各种辅助线、倒计时、测距标尺等（取决于你当前使用的武器）。
- **只有开启瞄准辅助后，特殊武器助手才会工作**。

### 4.3 开启辅助压枪
- 点击 **“开启辅助压枪”** 按钮或快捷键。
- 当你手持步枪等自动武器射击时，助手会自动帮你向下拉鼠标，减少后坐力。
- 首先需要打开装备栏识别当前的两个主武器以及其对应的配件，会显示在左下角人物血条栏上方，进而调整压枪力度
- 退出装备栏后会识别当前手持的武器。仅能识别特殊武器和装备栏有的主武器。
- 可以根据人物血条左边位置识别当前姿势，调整压枪力度。
- 手持连狙时，会有单发后坐力补偿，点击一下压一下枪
- 使用 M16A4 或 MK47 时，长按会自动进行连点，可单发变自动。
- 压枪只在武器检测成功且武器不是火箭筒、手雷、VSS、十字弩、C4 时生效。

## 5 特殊武器助手使用指南

这些助手只在你手持特定武器并且**瞄准辅助开启**时才会生效。你可以手动在界面上点击对应按钮来临时关闭某个助手。

### 5.1 迫击炮
- **生效条件**：瞄准辅助开启（不要求特定武器）。
- **用法**：
  1. 标点目标。
  2. 装备迫击炮并进入瞄准状态。
  3. 根据显示层小地图上方下层的四个色块中标注的距离，选择目标并调整距离，开炮

### 5.2 火箭筒（RPG-7 / 铁拳）
- **生效条件**：手持 Rocket。
- **用法**：
  1. 标点目标。
  2. 切换到第一人称腰射，不能开镜
  3. 屏幕上出现横线指示准星应该抬高的高度。
  4. 按标尺抬高瞄准点。

### 5.3 投掷物（雷、火、闪、篮圈）
- **生效条件**：手持投掷物。
- **用法**：
  1. 标点目标。
  2. 屏幕上出现横线指示准星应该抬高的高度。
  3. 按标尺抬高瞄准点。
  4. 若使用手榴弹，则按下V可根据标点距离自动拉环、丢出（自动瞬爆）
- **注意**：只能在显示层开启时使用。

### 5.4 VSS
- **生效条件**：手持 VSS。
- **用法**：
  1. 标点目标。
  2. 屏幕上出现横线指示准星应该抬高的高度。
  3. 按标尺抬高瞄准点。
- **注意**：只能在显示层开启时使用。

### 5.5 十字弩
- **生效条件**：手持十字弩。
- **用法**：
与vss相同
- **注意**：只能在显示层开启时使用。

### 5.6 C4
- **生效条件**：手持 C4。
- **用法**：
  1. 在车辆上，按下**鼠标左键**安装。
  2. 屏幕上显示“C4 安装中...”，持续 4 秒。
  3. 安装完成后，屏幕中央出现：
     - 爆炸倒计时（秒）
     - 建议车速（km/h）
     - 当前使用的标点颜色
  4. **切换标点颜色**：按 `Q` 或 `E` 键，可以在黄、橙、蓝、绿四种颜色中循环切换。助手会测量与你所选颜色标点的距离。
  5. **推荐起步**：当建议车速高于 70 km/h 时，会出现绿色“推荐起步”提示（持续 2 秒）。这意味着你应该起步冲向敌人了。
  6. **推荐跳车**：当你与标点的距离小于 20 米时，会出现红色“推荐跳车”提示（持续 2 秒）。这提醒你距离目标位置太近，应该立即跳车逃跑。

## 6 其他辅助功能

### 6.1 地图点位助手
- **用法**：按下 `鼠标左键` + `鼠标中键`，会显示该地图的**载具、飞机、密室**等点位
- 右键可取消显示，但若按住alt再按右键可不取消（对应游戏中标记路线）

### 6.2 大地图测距
- **默认快捷键**：`F1`（可在按键设置中修改）。
- **用法**：
  1. 按 M 打开大地图。
  2. 按测距快捷键，鼠标指针变为十字。
  3. 左键点击起点，会自动寻找到四个颜色的标点（如果存在）。
  4. 退出大地图，显示层中小地图上方上层四个距离为大地图测得距离。

### 6.3 切换武器标点颜色（仅限 C4）
- `Q` 和 `E` 键循环切换。

## 7 自定义快捷键

1. 点击 **“按键设置”** 选项卡。
2. 找到你想修改的功能（例如“手雷瞬爆”）。
3. 点击该行右侧的 **“录制”** 按钮。
4. 按下你希望的新快捷键。
5. 点击 **“保存快捷键”** 按钮。
6. 重启助手（或重新加载配置）后生效。

**注意**：手雷瞬爆和打开装备栏只允许单键（不能带 Ctrl、Shift、Alt）。选项卡中没有的快捷键不允许修改

## 8 状态栏解读

屏幕左下角的半透明窗口会显示以下信息：

| 行 | 内容 | 颜色含义 |
|----|------|----------|
| 第一行 | 识别、测距、压枪 | 绿色=功能正常，红色=关闭，黄色=压枪开启但无武器 |
| 第二行 | 装备栏状态 | 绿=已打开，蓝=关闭，橙=确认中 |
| 第三行 | 武器1名称及配件 | 白色文字 |
| 第四行 | 武器2名称及配件 | 白色文字 |
| 第五行 | 当前武器 + 姿势 | 站立/蹲下/趴下 |

这个覆盖层始终显示，即使你关闭了瞄准辅助。

## 9 常见问题（用户向）

### Q：我按了快捷键没反应？
- 检查瞄准辅助是否已开启（按钮状态为“关闭瞄准辅助”才是开启）。
- 确保你没有同时按下其他冲突的键。
- 尝试用鼠标点击一下游戏窗口，确保游戏窗口处于焦点。

### Q：C4 助手不显示倒计时？
- 确保手持 C4 并且瞄准辅助已开启。
- 必须**按住鼠标左键**直到安装进度条走完（4秒），不要提前松开。
- 如果还是不行，检查小地图标点是否已打上（Alt+鼠标右键）。

### Q：压枪时鼠标乱飘或不动？
- 重新校准准星区域。
- 关闭游戏内的“垂直灵敏度”增强功能。
- 尝试点击“重新加载压枪配置”按钮。

### Q：武器名称识别错误？
- 进入训练场，打开装备栏（Tab），然后在校准区域中重新校准“武器1名称”和“武器2名称”，选框只框住文字部分。
- 确保游戏语言为中文。

### Q：助手突然消失或闪退？
- 检查是否有杀毒软件拦截了助手。请将助手文件夹加入白名单。
- 如果还是闪退，尝试以管理员身份运行。

### Q：如何完全重置所有设置？
- 关闭助手，删除程序文件夹下的 `config.json` 文件，然后重新启动助手。所有校准和快捷键将恢复默认。
